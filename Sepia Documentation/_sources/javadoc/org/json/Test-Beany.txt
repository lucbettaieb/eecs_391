Test.Beany
==========

.. java:package:: org.json
   :noindex:

.. java:type:: static class Beany implements JSONString
   :outertype: Test

   Beany is a typical class that implements JSONString. It also provides some beany methods that can be used to construct a JSONObject. It also demonstrates constructing a JSONObject with an array of names.
